# Used Car Data Cleaning and Baseline Model

## Objective
To clean a messy used car dataset and evaluate a baseline model using Mean Absolute Error (MAE).

## Steps Performed
- Data exploration and issue identification
- Data cleaning (handling missing values, duplicates, formatting)
- Baseline model using mean selling price
- MAE evaluation

## Result
Baseline MAE: 551116.1133220675

## Tools Used
- Python
- Pandas
- Scikit-learn
